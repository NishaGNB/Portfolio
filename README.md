# Portfolio2

A minimal static portfolio project demonstrating a single-page layout built with HTML, CSS, and JavaScript.

## Overview

This project contains a simple portfolio page (`index.html`) with associated styling and client-side scripting. It's intended as a lightweight, easy-to-run example you can open in any modern browser.

## Files

- [index.html](index.html) : Main HTML page for the portfolio.
- [style.css](style.css) : Styles for layout and appearance.
- [script.js](script.js) : Client-side behavior and interactivity.

## How to run

1. Open [index.html](index.html) in your web browser (double-click or right-click â†’ Open with).
2. Alternatively, serve the folder using a simple static server (recommended for some features):

   - Python 3:

```bash
python -m http.server 8000
```

   - Node (http-server):

```bash
npx http-server -c-1
```

Then open http://localhost:8000 in your browser.

## Customization

- Edit `index.html` to change content and structure.
- Modify `style.css` to adjust layout, colors, and typography.
- Update `script.js` to change interactive behavior.
